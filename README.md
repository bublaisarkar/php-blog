📌 Overview

This is a dynamic Blog Management System built using Core PHP and MySQL.
The system provides user authentication, blog CRUD operations, category management, and admin panel features.

This project is developed as a portfolio project to demonstrate backend development skills and real-world deployment experience.

📂 Project Structure
/blog-project

│── _db/            → Database SQL files  
│── admin/          → Admin dashboard pages  
│── ajax/           → AJAX request handlers  
│── css/            → Stylesheets  
│── img/            → Static images  
│── inc/            → Common includes (header, footer, config)  
│── js/             → JavaScript files  
│── php/            → Backend processing scripts  
│── upload/         → Uploaded blog images  

│── index.php       → Homepage  
│── blog.php        → Blog listing page  
│── blog-view.php   → Single blog view  
│── Category.php    → Category filtering  

│── login.php       → User login  
│── signup.php      → User registration  
│── logout.php      → Logout  

│── admin-login.php → Admin authentication  
│── db_conn.php     → Database connection  

⚙️ Local Installation Guide
Install XAMPP / WAMP / LAMP
Copy project folder into htdocs
Create a MySQL database
Import SQL file from _db folder
Update database credentials in db_conn.php
Run project:
http://localhost/blog-project/
